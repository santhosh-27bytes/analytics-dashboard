## 📖 About the Project

**Analytics Pro** is a fully interactive, client-side web application designed to analyze student academic performance from Excel/CSV data.

It enables educators and students to:

* Visualize trends
* Compare performance
* Generate rankings
* Access detailed report cards

—all within a clean, responsive interface without requiring a backend.

---

## ✨ Core Features

### 📊 Class Overview

* Subject-wise average performance visualization
* Dynamic color-based insights
* Supports **Bar & Line charts**

### 👥 Student Comparison

* Compare multiple students simultaneously
* Input via Roll No / ID
* Supports **Bar & Radar charts**

### 📚 Subject Analytics

* Select and analyze specific subjects
* Displays:

  * Average performance
  * Pass/Fail percentages
* Interactive insights (click to view student lists)

### 🏆 Ranking System

* Automated ranking based on performance
* Intelligent grade mapping (O, A+, A, etc.)
* Pass/Fail classification

### 📄 Student Report Card

* Search by Name / Roll No / ID
* Detailed subject-wise breakdown
* Grade + percentage visualization

### 📁 File Upload System

* Supports:

  * Excel (`.xlsx`, `.xls`)
  * CSV (`.csv`)
* Instant parsing and dashboard update
